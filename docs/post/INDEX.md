## üéØ Feature Reference

### Post List Page (`/post/list`)
- Search by title
- Filter by status & date
- Pagination
- Row selection
- Bulk delete
- CRUD actions
- Loading states

### Post Detail Page (`/post/:id/detail`)
- Display post info
- Show thumbnail
- View categories
- Read content
- Edit/Delete actions

### Post Create/Edit Page (`/post/create`, `/post/:id/edit`)
- Title input (3-200 chars)
- Description (10-500 chars)
- Thumbnail upload
- Categories (multi-select)
- Publish date/time
- Content field (100+ chars)
- Visibility selection
- Full validation

### API Functions
1. `getPostList()` - List with filters
2. `getPostById()` - Get single post
3. `createPost()` - Create new
4. `updatePost()` - Update existing
5. `deletePost()` - Delete single
6. `deletePostsBulk()` - Bulk delete
7. `getCategories()` - Get categories
8. `publishPost()` - Publish post
9. `archivePost()` - Archive post
